# 2026-03-18 세션 요약

## 작업 내용

### 1. 라운드 표시 UI
- `CombatHUD`에 `m_RoundText` (TextMeshProUGUI) 필드 추가
- `OnRoundStarted(RoundStartedEvent e)`에서 `m_RoundText.text = e.Round.ToString()` 호출
- 전투 씬 중앙 상단에 숫자만 표기

### 2. 라운드 텍스트 팝인 애니메이션
- `RoundUI` 구조: roundbg1 / round_effect (Image + Animator + RoundAnimEvent.cs) / roundbg2 / roundtext
- `round_effect` SetActive false → true 토글로 매 라운드 팝인 재트리거
- `CombatHUD.ShowAllTickersAnimated()` 내 ticker 활성화 직후에 토글 삽입 → ticker와 동일 타이밍
- **Animation Event 브릿지 패턴** 적용:
  - 애니 클립 0.3초 지점에 Event 추가, Function: `ApplyRoundText`
  - `RoundAnimEvent.cs`를 round_effect에 부착, `m_CombatHUD` 참조
  - `CombatHUD.ApplyRoundText()` — `m_PendingRound` 저장 후 이벤트 타이밍에 텍스트 갱신
  - GC 절감을 위해 코루틴 대신 Animation Event 방식 채택

### 3. 적 타겟 하이라이트
- NikkeSlot 각각에 `EnemyTargetHighlight (Image)` 추가 (기존 Highlight와 별개, 색상 다름)
- `CombatHUD`에 `m_NTargetHighlights` (Image[]) 필드 추가
- `public void ShowEnemyTargetHighlight(int slotIndex)` — 전체 hide 후 해당 슬롯 활성화
- `public void HideEnemyTargetHighlights()` — 전체 비활성화
- `HandleEnemyTurn()`에서 `DecideAction` 직후 Show, 두 번째 딜레이 직후 Hide

### 4. 적 스킬명 표시 UI (AnnouncePanel)
- `TurnOrderPanel` 아래 `AnnouncePanel (empty)` 추가 — Animator + Image + TMP text
- `CombatHUD`에 `m_EnemySkillPanel` (GameObject), `m_EnemySkillNameText` (TMP) 필드 추가
- `public void ShowEnemySkillName(string skillName)` — 텍스트 설정 후 SetActive false → true (팝인)
- `public void HideEnemySkillName()` — SetActive false
- `action.IsPass`일 때 `"턴 넘김"` 표시
- `HandleEnemyTurn()`에서 타겟 하이라이트와 동일 타이밍에 호출

## 설계 결정
- **Animation Event vs 코루틴**: GC 절감 목적으로 Animation Event 채택
- **브릿지 스크립트 패턴**: Animator GameObject ≠ 로직 스크립트 GameObject일 때 사용
- **팝인 재트리거**: SetActive false → true 토글 방식 (Ticker와 동일 패턴)
